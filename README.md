## eligibility-api
Analyze the information available in the consumption history and tariff modes to define prospect eligibility

## Docs
Run the app then to /docs

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Running the app (with docker-compose)
```bash
docker-compose up
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
